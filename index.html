Aprenda de uma vez por todas as diferenças entre as linguagens de programação!

Primeiramente é importante ressaltar que não será analisado todas as lingugens de programação por conta da enorme diversidade.

Contudo, separei 6 linguagens: Javascript, Java, PHP, Python, C# e Assembly para analisar e mostrar um pouquinho de suas particularidades.

Linguagem Ano de criação Nivel IDE Tipo Opinião
JavaScript 1995 alto nível Visual Studio Code (VS Code) e WebStorm Front-end Tenho pouca experiência com javascript, mas a priori sua sintaxe não é tão complexa quanto java, mas também não é tão simples quanto python
Java 1995 alto nível IntelliJ IDEA, Eclipse e NetBeans Back-end Conheço pouco de Java, apesar de já ter feito algumas atividades, contudo sua sintaxe é complexa comparada a Python, e pra mim foi difícil realizar os códigos
PHP 1995 alto nível PhpStorm e Visual Studio Code back-end Particularmente não conheço muito pois, estou iniciando em desenvolvimento web, mas sei que ela é para realizar a parte de back-end em aplicações web
Python 1991 alto nível/td> PyCharm,Visual Studio Code e Jupyter Notebook back-end Particularmente é minha linguagem favorita.Tive a oportunidade de conhecer em 2021 e de lá pra cá, de todas que eu vi até agora, ainda prefiro Python por sua simplicidade e praticidade
C# 2000 alto nível Visual Studio back-end Ainda não tive a oportunidade de trabalhar com essa linguagem, mas sei que uma de suas aplicabilidades é no desenvolvimento de aplicativos para o SO Windows
Assembly 1949 baixo nível MARS back-end Nunca tive vivência com assembly, mas sei que é uma das linguagens mais antigas que existem e sua aplicabilidade é para escrever programas que interagem diretamente com o hardware de um computador
